Introduction

This website hosts exercises for the HTML, CSS, and Javascript course created by Johns Hopkins University and hosted by Coursera.

Module 2

The website in Module 2 is a basic demonstration of 'responsive design'. Using CSS, it allows for dynamic resizing of content such that different screen sizes and resolutions are accommodated. Theoretically, the site in module 2 can be viewed equally well on mobiles, tablets, and full-size computers.

Module 3

The website in Module 3 is a further demonstration of 'responsive design'. Using the Twitter Bootstrap framework, it adds a navigation bar. In mobile view, the navigation bar reveals a dropdown menu that allows better access to the rest of the page.

Module 4

Module 4 is an exercise in Javascript. It consists of a loop over an array of names, printing out 'Hello' or 'Good Bye' depending on the first letter of the name.
